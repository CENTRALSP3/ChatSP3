# Vamos Interagir Project

Front‑end and back‑end skeleton for the **“Vamos Interagir”** real‑time interaction app.

> This repo contains minimal boilerplate to start developing a Vite + React + TypeScript front‑end
> and an Express + Socket.io back‑end. All state lives in memory – no database.

## Structure
- `/backend` – Express server + Socket.io logic.
- `/frontend` – Vite + React + Tailwind + Framer Motion.
- `/common` – shared types used by both sides.

Run `npm i` in each folder before starting.
